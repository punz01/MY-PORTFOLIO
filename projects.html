My Projects

Project 1: Car Guys Cave

For my project on Dynamic Web Applications, I utilized Wordpress to build a website showcasing Japanese Domestic Market cars, detailing their names, histories, prices, and engine specifications. Despite the initial challenge, as I designed more pages, I found enjoyment and comfort in using Wordpress.

Project 2: Raw Photography

In the Web Search Engine Optimization and Analytics course project, we're concentrating on a photography website, publishing blogs to boost engagement. Using Wordpress again, I'm tasked with designing and arranging pages to improve search engine ranking and analytics.

Project 3: Kings of the Court Camp

In my initial website project for Web Technologies, users can enroll in home basketball training, accessing tutorials and schedules. Developed in Visual Studio Code using HTML & CSS, I encountered language challenges at first but have since been steadily mastering new CSS properties with each project.

Project 4: Victory Fit

This collaborative project merges Dynamic Web Applications, Development Tools, and Information Management, focusing on a Gym website. Users can sign up, select coaches for personalized workouts and schedules, and purchase supplements from our store. Developed in Visual Studio Code, utilizing HTML, CSS, PHP, and MySQL, my role in the group is to enhance frontend engagement.

Project 5: PPAR Review

Inspired by Rotten Tomatoes, our project in Advanced Database Systems centers on anime ratings. Users can rate and review within selected genres after signing up. Developed in Visual Studio Code, using HTML, CSS, PHP, and MySQL, I'm responsible for crafting each page's frontend design.
